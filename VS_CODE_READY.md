## 🚀 Quick Start

### 1. Install VS Code Extensions (Recommended)

Open VS Code and install these recommended extensions:
- ESLint (`dbaeumer.vscode-eslint`)
- Prettier (`esbenp.prettier-vscode`)
- Tailwind CSS IntelliSense (`bradlc.vscode-tailwindcss`)
- ES7+ React/Redux/React-Native snippets (`dsznajder.es7-react-js-snippets`)

**Quick Install:** VS Code will prompt you to install recommended extensions when you open the project.

### 2. Install Dependencies

```bash
npm install
# or
pnpm install
# or
yarn install
```

### 3. Start Development Server

```bash
npm run dev
# or
pnpm dev
# or
yarn dev
```

The app will open at: **http://localhost:5173**

## 💡 Development Tips

### TypeScript
- VS Code will show type errors in real-time
- Hover over variables to see types
- Use `Ctrl+Space` for auto-completion
- Use `F12` to go to definition

### React Components
- Use `.tsx` extension for React components
- Import React automatically with `jsx: "react-jsx"`
- Use functional components with hooks

### Styling
- Tailwind IntelliSense will suggest classes
- Use custom CSS variables in `src/styles/theme.css`
- Responsive design with `sm:`, `md:`, `lg:`, `xl:` prefixes

### Debugging
- Use browser DevTools
- React DevTools extension
- Console.log strategically
- Check Network tab for API calls

## 🛠️ Troubleshooting

### TypeScript Errors
If you see TypeScript errors:
```bash
npm run type-check
```

### ESLint Warnings
To fix auto-fixable issues:
```bash
npm run lint -- --fix
```

### Format All Files
```bash
npm run format
```

### Clear Node Modules
If dependencies are causing issues:
```bash
rm -rf node_modules
npm install
```

### VS Code Not Detecting TypeScript
1. Open Command Palette (`Cmd/Ctrl + Shift + P`)
2. Type "TypeScript: Select TypeScript Version"
3. Select "Use Workspace Version"
