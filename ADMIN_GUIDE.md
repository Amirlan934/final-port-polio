# Portfolio Admin Dashboard

## Admin хэрэглэх заавар

### Нэвтрэх

1. Вебсайтын navigation дээрх **Settings (⚙️)** товч дээр дарна
2. Эсвэл шууд `/admin` хаягаар очино
3. Нууц үг оруулна: `admin123`

### Төслүүд (Projects) удирдах

- **Нэмэх**: "Төсөл нэмэх" товч дарж шинэ төсөл үүсгэнэ
- **Засах**: Төсөл дээр "Edit" (✏️) товч дарж засна
- **Устгах**: "Delete" (🗑️) товч дарж устгана
- Төсөл бүр дараах мэдээлэл агуулна:
  - Гарчиг (title)
  - Ангилал (category)
  - Тайлбар (description)
  - Зургийн URL
  - Tags (шошго)
  - Featured эсэх (үндсэн хуудсанд харуулах эсэх)

### Үйлчилгээ (Services) удирдах

- Services хэсэгт үйлчилгээнүүдээ нэмж, засаж болно
- Үйлчилгээ бүр гарчиг, тайлбар агуулна

### Сэтгэгдэл (Testimonials) удирдах

- Үйлчлүүлэгчдийн сэтгэгдэл нэмэх, засах, устгах
- Нэр, албан тушаал, компани, сэтгэгдлийн агуулга, зураг оруулна

### Hero хэсэг засах

- Нүүр хуудасны үндсэн контентыг засна:
  - Мэндчилгээ текст
  - Гарчиг
  - Дэд гарчиг
  - Тайлбар
  - Товчны текст
  - Портрет зураг
  - About Me мэдээлэл

### Өгөгдөл хадгалах

- Бүх өөрчлөлт автоматаар **localStorage** дээр хадгалагдана
- Хөтөч refresh хийсэн ч өгөгдөл алдагдахгүй
- "Анхны төлөвт буцаах" товч дарж өгөгдлийг reset хийж болно

### Техникийн мэдээлэл

- Frontend-only solution (backend байхгүй)
- localStorage ашигласан (5-10MB багтаамж)
- React Context API ашигласан state management
- Нууц үг: `admin123` (localStorage дээр хадгална)

### Анхааруулга

⚠️ Энэ нь demo/prototype admin dashboard юм. Production хувилбарт:
- Жинхэнэ backend database ашиглах (PostgreSQL, MongoDB гэх мэт)
- Аюулгүй authentication систем (JWT, OAuth)
- Зураг upload үйлчилгээ (AWS S3, Cloudinary)
- Админ эрхийн удирдлага
- Өгөгдлийн нөөцлөлт

## Тусламж

Асуудал гарвал:
1. Browser console-ыг шалгана (F12 дарна)
2. localStorage-ын өгөгдлийг устгаж үзнэ
3. "Анхны төлөвт буцаах" товч дарна
