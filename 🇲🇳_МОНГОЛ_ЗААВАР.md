## 🚀 ХЭРХЭН ЭХЛҮҮЛЭХ ВЭ?

### 1. Dependencies суулгах

```bash
npm install
```

эсвэл

```bash
pnpm install
```

### 2. Development server эхлүүлэх

```bash
npm run dev
```

Вебсайт нээгдэнэ: **http://localhost:5173**

### 3. VS Code дээр нээх

```bash
code .
```

VS Code автоматаар extension-үүдийг суулгахыг санал болгоно.
